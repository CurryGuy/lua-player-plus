========================================================================
PROGETTO MAKEFILE: cenni preliminari sul progetto Lua Player Plus
========================================================================

La creazione guidata applicazione ha creato questo progetto 
Lua Player Plus.  

Questo file contiene un riepilogo del contenuto di ciascun file che fa 
parte del progetto Lua Player Plus.


Lua Player Plus.vcxproj
    File di progetto principale per i progetti VC++ generati tramite una 
    creazione guidata applicazione. 
    Contiene informazioni sulla versione di Visual C++ che ha generato il 
    file e informazioni sulle piattaforme, le configurazioni e le 
    caratteristiche del progetto selezionate con la Creazione guidata 
    applicazione.

Lua Player Plus.vcxproj.filters
    File dei filtri per i progetti VC++ generati tramite una Creazione 
    guidata applicazione. 
    Contiene informazioni sull'associazione tra i file del progetto e i 
    filtri. Tale associazione viene utilizzata nell'IDE per la visualizzazione 
    di raggruppamenti di file con estensioni simili in un nodo specifico, ad 
    esempio: i file con estensione cpp sono associati al filtro "File di 
    origine".

Il progetto consente di compilare/pulire/ricompilare da Visual Studio chiamando 
i comandi specificati nella procedura guidata. Il comando di compilazione può 
essere nmake o un qualsiasi altro strumento in uso.

Il progetto non contiene file. Nessun file visualizzato in Esplora soluzioni.
